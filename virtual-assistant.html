Virtual Assistant

A program can do basic assistance using voice or text commands.

SUMMARY

[!]THIS PROJECT IS A WIP[!] The virtual assistant NLP input speech to text. It then uses a pre defined list of commands and checks if the input matches a command, in which case the command is executed. The assistant can most basic activities such as opening a website, playing songs, opening a location on maps, etc. This is merely a project that I work on during my free time, hence progress will be slow.

Technologies Used

Pyttsx3 - Python library used to convert human speech to text

Python- The complete code is built entriely on Python
