Twitter Sentiment Analysis

A simple CLI tool which analyzes the sentiment on a topic.

SUMMARY

This is a simple CLI tool which analyzes the sentiment on a topic. Using the twitter API, the tool fetches the latest tweets on a topic inputted by the user and then analyzes the sentiment of each tweet. It then generates a pie chart report showing the overall sentiment.

Technologies Used

Twitter API

Python- The complete code is built entriely on Python
